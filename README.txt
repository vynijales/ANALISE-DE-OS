Projeto Individual:
•	Desenvolvi um software para a padronização e facilitação de análises de atendimento nível 2 de ordens de serviço na empresa SISTEMA OESTE DE COMUNICAÇÃO LTDA.
•	Permite aos usuários personalizarem suas análises com base em scripts, selecionando itens através de uma interface gráfica e ajustando-os de acordo com suas necessidades, mas mantendo uma estrutura padronizada.
•	Com um simples clique no botão "copiar", todos os resultados são transferidos rapidamente para a área de transferência.
•	O software foi desenvolvido em Python e o transformado em arquivo executável.
•	Atualmente, ainda é utilizado e tem contribuído para a eficiência e qualidade das análises na empresa.
